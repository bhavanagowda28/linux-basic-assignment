# Linux Basics Assignment

## Objective
This assignment demonstrates basic Linux commands.

---

## 1. File and Directory Operations
mkdir test_dir  
touch test_dir/example.txt  
mv test_dir/example.txt test_dir/renamed_example.txt  

---

## 2. Viewing Files
cat /etc/passwd  
head -5 /etc/passwd  
tail -5 /etc/passwd  

---

## 3. Searching
grep "root" /etc/passwd  

---

## 4. Compression
zip -r test_dir.zip test_dir  
unzip test_dir.zip -d unzipped_dir  

---

## 5. File Download
wget -O sample.txt https://www.gnu.org/licenses/gpl-3.0.txt  

---

## 6. Permissions
touch secure.txt  
chmod 444 secure.txt  

---

## 7. Environment Variable
export MY_VAR=Hello,\ Linux!  
echo $MY_VAR
